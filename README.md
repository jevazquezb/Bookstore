## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS

### Setup

    1. git@github.com:jevazquezb/Bookstore.git
    2. cd Bookstore

### Install

    npm install

### Usage

    npm start
